# COVID Futures

Predictions about Australia's COVID future.


## Getting Started

Install pipenv:

```
pip install pipenv
```

Install dependencies (both Python and JS):

```
pipenv install
npm install
```

Build assets:

```
npm run webpack:dev
# or
npm run webpack:prod
```

Run the development server:

```
pipenv run server
```

Run tests:

```
pipenv run test
# or
pipenv run test-watch
```
